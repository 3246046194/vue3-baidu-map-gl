## Get started

### installation
```shell
npm i vue3-baidu-map-gl
# or
yarn add vue3-baidu-map-gl
# or
pnpm add vue3-baidu-map-gl
```

### Usage
```vue
<template>
	<Map ak="YOUR_AK" />
</template>

<script setup lang="ts">
  import Map from 'vue3-baidu-map-gl';
</script>
```

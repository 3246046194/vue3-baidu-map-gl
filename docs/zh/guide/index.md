基于百度地图 JavaScript GL 版 API 封装的 Vue3 组件库。

## :sparkles: 特性

-   自动加载百度地图 SDK，你需关注组件使用。
-   使用 Typescript 编写，完善度组件代码提示。
-   不依赖任何第三方组件。
-   组件同时兼容大驼峰命名(CamelCase)和短横线命名(kebab-case)
